# Events Operating System

**The AI-Native Operating System for the events industry.**
Built by [Reality Near](https://realitynear.org) · Founded by [Javier Bambarén Dabdoub](https://jbd84.github.io)

---

## What is EventOS?

EventOS replaces the operational chaos of spreadsheets, WhatsApps, and disconnected tools with a single AI-native platform that humans and AI agents can both understand and operate.

Built by someone who ran events for 20+ years — not by someone who read about the problem.

> *"En un mundo donde todo es programable, tu imaginación es el único límite."*
